## Smart Scroll Container (hides items outside viewport).

Usage:
```jsx
import SmartScrollContainer from 'react-smart-scroll-container';

<SmartScrollContainer avgItemHeight={100} initialVisibleCount={10}>
    <div>ITEM1</div>
    <div>ITEM2</div>
    <div>ITEM3</div>
    <div>ITEM4</div>
</SmartScrollContainer>
```
